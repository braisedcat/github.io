去除chrome新标签页缩略图（most visited pages)

chrome v61及之前版本请参考知乎问题怎么去掉 Chrome 新标签页的缩略图？

chrome v62

  1. 下载工具：ChromePak V5 ,感谢作者。ps：网站可能被墙

  2. 解压，定位至对应版本的pak_tools目录下（windows为386），复制粘贴chrome的resources.pak（一般在C:\Program Files (x86)\Google\Chrome\Application\62.x.x.x下）。

  3. 解包：

    unpack files in resources.pak to resources.json and resources folder

  4. 修改文件\resources\chrome\browser\resources\local_ntp\most_visited_single.css。

    在#most-visited下添加代码：display: none !important;

  5. 重新封装resources.pak ：

    repack files to resources.pak according to resources.json

  6. 替换原来的resources.pak文件。
