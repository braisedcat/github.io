vps基本设置

FreeTime
2017-10-01

环境

  • bandwagon 10G KVM vps
  • 安装新OS:Ubuntu 16.04 x86_64,操作比较熟悉

添加ipv6

  1. 注册 tunnelbroker,然后点击Create Regular Tunnel,接着输入vps地址,选择地址来创建ipv6隧道.
  2. 点击Example Configurations,选择OS:Debian/Ubuntu.复制框里内容粘贴至vps的/etc/network/interfaces文件末尾.
  3. 启用: ifup he-ipv6

SS

  1. 安装

  2. 配置
    vi /etc/shadowsocks.json
    粘贴以下内容,修改 server_port 和 password内容.server不用修改,支持ipv6

    {
 "server":"::", 
 "server_port":1234,
 "local_address": "127.0.0.1",
 "local_port":1080,
 "password":"password",
 "timeout":300,
 "method":"aes-256-cfb",
 "fast_open": false
}
  3. 添加开机启动
    vi /etc/rc.local
    在exit 0前添加启动命令:
    ssserver -c /etc/shadowsocks.json -d start
    ifup he-ipv6
